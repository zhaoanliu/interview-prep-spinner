# Daily Interview Prep Spinner

A lightweight, single-file web app for daily interview prep focus.
Built as a quick trial project to explore Cursor AI-assisted coding.

## What it does

Spin three wheels to set your interview prep focus for the day:

- **LeetCode Category** — picks from 15 algorithm topics (NeetCode 150 roadmap)
- **System Design Topic** — picks from 10 classic system design problems
- **Behavioral Question** — picks from common behavioral interview prompts

## How to use

Just open `prep-spinner.html` in any browser. No install, no dependencies.

## Tech

Pure HTML, CSS, and JavaScript — single file, no frameworks, no build step.

## Background

This was a 1-hour trial project to get hands-on with
[Cursor](https://cursor.com), an AI-powered code editor.
The entire app was generated and iterated using Cursor's
Composer (Agent mode) with a few natural language prompts.
